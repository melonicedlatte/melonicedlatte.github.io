## 1. 문제

https://www.acmicpc.net/problem/5585

## 2. 풀이 방법

가장 비싼 값을 가진 코인부터 가진 돈에서 빼주면서 개수를 구하면 항상 최소 동전의 개수가 나옵니다. 

## 3. 소스코드

~~~python
coin = [500, 100, 50, 10, 5, 1]
_input = 1000 - int(input())
coin_num = 0
for _coin in coin:
    now_coin = (int) (_input / _coin)
    if now_coin > 0 :
        _input = _input - (now_coin * _coin)
        coin_num += now_coin

print(coin_num)
~~~

문제

영어 대소문자와 띄어쓰기만으로 이루어진 문장이 주어진다. 이 문장에는 몇 개의 단어가 있을까? 이를 구하는 프로그램을 작성하시오.

입력

첫 줄에 영어 대소문자와 띄어쓰기로 이루어진 문장이 주어진다. 이 문장의 길이는 1,000,000을 넘지 않는다. 단어는 띄어쓰기 한 개로 구분되며, 공백이 연속해서 나오는 경우는 없다.

출력

첫째 줄에 단어의 개수를 출력한다.

예제 입력 1 

The Curious Case of Benjamin Button

예제 출력 1 

6

  • 빠진 조건을 찾은 사람: his130

알고리즘 분류

풀이

c++ 과 c언어는 string 과 char [] 에 대한 제대로 된 이해가 되지 않은 것 같습니다.
그냥 무식하게 풀었습니다. 

이 문제는 쉬워 보이나 정답률이 높지 않습니다.
질문게시판에 jh05013 님이 올려주신 글을 보면 왜 틀리는지 알 수 있습니다.

  1. 맨 처음에 공백이 들어올 수도 있습니다.
  2. 맨 끝에 공백이 들어올 수도 있습니다.
  3. 아예 공백만 있을 수도 있습니다.
  4. 하지만 공백이 둘 이상 연속으로 오지는 않습니다.
  5. C의 char[]로 문자열을 저장하고자 한다면 널문자를 포함하여 최대 1,000,001글자를 받아야 합니다.

위의 조건들을 놓치면 틀릴 수 있는 문제입니다. 

python 이면 1줄이면 끝나지만, c언어는 제 지식에서는 다소 복잡했습니다. 

먼저 모든 문자열을 char배열에 받는다.
그 다음으로 앞의 문자를 받아서 문자열이 아니면 무시하고 인덱스를 넘겨준다.
그리고 그 인덱스부터 탐색하여 개행문자 ' ' 를 만나면 그 다음 문자가 문자열인지 확인하고 문자열이면 1을 더해주고 아니면 그냥 넘긴다.

이런 식으로 반복하여 문제를 해결했다.

#include <iostream>
#include <vector>
#include <algorithm>
#include <functional>
#include <string.h>
#pragma warning(disable : 4996)
using namespace std;
 
#define MAX 1000000 + 2 
char ARR[MAX];
 
int main() {
    fgets(ARR, MAX, stdin);
 
    string _string(ARR);
    int str_len = _string.length();
    int answer = 1; 
    int s_idx = 0 ;
    // 문자 열의 길이가 끝날 때 까지 문자인지 확인하고 아니면 index를 늘려준다. 
    while (s_idx < str_len)  { 
        int _little_status = 'a' <= _string.at(s_idx) && 'z' >= _string.at(s_idx);
        int _big_status = 'A' <= _string.at(s_idx) && 'Z' >= _string.at(s_idx);
        if (!(_little_status || _big_status))
            s_idx++;
        else
            break;
    }
 
    // 만약 문자 열의 끝까지 아무글자도 없으면 0을 출력하고 리턴 
    if (s_idx == str_len) {
        cout << 0;
        return 0; 
    }
 
    // 그 다음부터 문자열을 탐색한다.
    // 해당 인덱스의 값이 ' ' 이면 다음 문자열의 값이 문자열인지 확인한다. 
    // 문자열이라면 문자열의 수를 1개 추가시켜 준다. 
    for (; s_idx < str_len; s_idx++) {
        // ' ' 이면 갯수 하나 추가 
        if ( _string.at(s_idx) ==  ' ' ){
            int _little_status = 'a' <= _string.at(s_idx + 1) && 'z' >= _string.at(s_idx + 1);
            int _big_status = 'A' <= _string.at(s_idx + 1) &&  'Z' >= _string.at(s_idx + 1);
            if (_little_status || _big_status)
            answer++; 
        }
    }
 
    cout << answer;
 
    return 0;
}

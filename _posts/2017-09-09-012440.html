윈도우에서 환경 변수에 pip 를 추가해도 작동하지 않을 때가 있습니다.

그럴 때는

아래와 같은 명령어를 사용하면 쉽게 해결할 수 있습니다. 


python -m pip install ~~

혹은 파일 경로를 pip 대신에 넣어서 사용 할 수 있습니다. 

C:\WinPython-64bit-2.7.13.1Zero\python-2.7.13.amd64\Scripts\pip.exe

var con_test = confirm("어떤 값이 나올까요.");
if(con_test == true){
    alert("확인을 누르셨군요");
}
else if(con_test == false){
    alert("취소를 누르셨군요.");
}



위와 같은 소스 코드를 입력하면 alert가 아닌 confirm 창이 떠서 확인 취소를 누를 수 있다.

alert는 단순히 경고만 주기 때문에, 확인이나 취소를 누를 수 없다. 


해당 버튼을 누르면 발생하는 이벤트는 위와 같이 정의할 수 있다. 

자바 스크립트 코드 안에 저 코드를 넣어주면 간단하게 동작하게 할 수 있다.

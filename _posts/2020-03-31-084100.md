python에서 string의 단어(character)를 list로 만드는 방법은 아래와 같이 쉽게 할 수 있습니다.

## List Comprehension를 사용하는 방법

~~~python
def str_split(word): 
    return [char for char in word]  
      
print(str_split('test')) 
~~~

## list()를 사용하는 방법

~~~python
def str_split(word): 
    return list(word) 
      
print(str_split('test')) 
~~~

아마존 웹 서비스에 apache를 설치하고 접속을 시도했는데 되지 않더라구요.





아래의 명령어로 포트가 열려있나 확인 해 보았을 때, 80번 포트는 이상 없이 동작 중이었습니다. 

$ netstat -tupln





확인해보니 port 설정이 기본적으로 80 번이 열려 있지 않았기 때문에 그런 것이었습니다. 
기본 설정은 무조건 닫혀 있는 것 같습니다. 

aws 의 instance 설정에 가서 오른쪽으로 넘긴 후에 Security Groups 라는 탭의 하이퍼링크를 클릭합니다. 
여기서는 launch-wizard-1 가 되겠네요 




그 다음 Inbound 에 들어가서 Edit 버튼을 눌러줍니다. 





그 다음에는 Edit inbound rules 창에서 Add Rule 버튼을 눌러 준 다음에 HTTP 규칙을 추가해줍니다. 





그 다음 해당 ip 로 접속하면 완료 !!!

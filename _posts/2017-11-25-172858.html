라즈베리파이는 나무위키의 출처에 따르면 아래와 같이 설명하고 있습니다. 

영국 라즈베리 파이(Raspberry Pi) 재단에서 만든 초소형/초저가 PC이다. 교육용 프로젝트의 일환으로 개발[1]되었으며, 이 때문에 RCA(CVBS) 출력 잭을 가지고 있다. 2012년 3월에 출시되었는데, 1시간만에 매진되는 진풍경을 보여줬다. 2013년 11월 기준 200만대 이상이 팔렸다. 이런 이유로 인해서 몇몇 학교들에서는 라즈베리 파이를 교육용으로 이용한다. 아두이노와 함께 소수의 관련 업계 엔지니어들만의 영역이었던 개발 보드의 저가화와 대중화의 시대를 연 주역


라즈베리파이(Raspberry Pi)
– 2006년 영국 캠브리지 대학 에반업튼 박사가 개발 라즈베리파이 재단 설림
– 영국 라즈베리파이 재단(www.raspberrypi.org)이 학교와 개발도상국에 컴퓨터를 보급하기 위해 개발한 싱글보드 컴퓨터
– 저렴한 가격
    • Raspberry Pi 3 Model B 가격은 4만원대
– 신용카드 크기
– 컴퓨터로 동작할 만큼 충분한 성능

굉장히 저렴한 소형 컴퓨터이고 가격도 비싸지 않은 수준이라서 프로젝트나 교육용으로 많이 사용되고 있습니다. 

라즈베리파이로 아래와 같은 재미있는 프로젝트를 하기도 합니다. 
자금에 여유가 생기면 해봐야겠네요. 





저 같은 경우에는 편하게 사용하기 위해서 모니터가 달려있는 제품 세트를 구매하였습니다. 
http://www.eleparts.co.kr/EPXGP3KM

이 세트와 추가로 sd 카드, sd 카드 리더기가 필요합니다. 
sd카드는 기존에 있던 걸 사용했고, 리더기는 다이소에서 2천원 주고 구매했습니다.

일단 OS 를 설치한 다음에 연결을 해 보겠습니다. 



실제 구매한 내용은 아래와 같습니다. 

라즈베리파이를 사용하기 위해서 라즈베리파이3, 라즈베리파이 Touchscreen Display 및 기타 물품을 구매하였습니다. 


초기에는 OS 를 설치해야됩니다. 

OS 는 라즈베리파이 재단에서 만든 Raspbian을 사용하겠습니다. 

Raspbian
• Raspberry pi + Debian
• 데비안 기반의 라즈베리파이 운영체제
• 라즈베리 파이의 메인 OS로 라즈베리파이 재단에 의해 지원됨


https://www.raspberrypi.org/downloads/noobs/

해당 링크에 가서 파일을 다운받겠습니다. 

NOOBS is an easy operating system installer which contains Raspbian. It also provides a selection of alternative operating systems which are then downloaded from the internet and installed.

NOOBS 파일에 Raspbian 이 들어가 있기 때문에 해당 파일을 설치해줍니다. 



다운 받는 순간에 SD 카드를 세팅해주어야 합니다. 
Raspbian 을 설치할 때에는 SD카드 Format이 필요합니다.  (미리 되어 있는 경우는 할 필요 없음 )
Raspberry Pi3부터 USB 부팅도 가능하지만, SD카드를 사용하겠습니다. 

https://goo.gl/hxtjB







자신의 운영체제에 맞는 버전을 클릭하고, 하단의 accept 를 눌러줍니다. 



위의 링크에 들어가서 다운로드 받아줍니다. 

설치 과정에 맞춰서 설치를 끝내면, 아래와 같은 프로그램을 실행할 수 있습니다. 







그 다음 Flash Memory 에 파일을 옮겨 넣습니다.

폴더 안의 파일들을 옮겨주세요. 




SD카드를 삽입하면 아래의 그림에서 빨간불 오른쪽에, 노란불이 깜빡이면서 실행됩니다. 
마우스를 끼워주고 hdmi 를 연결해서 모니터를 연결해야합니다. 
아직은 7인치 LCD 를 사용하지 않기 때문에, 제 컴퓨터의 모니터를 사용하겠습니다. 







그 중, 위에 있는 설치 프로그램을 선택하고 위에 있는 Install 을 클릭합니다.

화면은 기본이 180도가 돌아가 있기 때문에 아래와 같은 설정이 필요합니다. 


화면의 위아래가 뒤집혀요!

디스플레이 스탠드에 따라서는 LCD 디스플레이의 위아래가 바뀐 것이 기본값일 수도 있습니다. 이때에는 /boot/config.txt에서 회전시킴으로써 문제를 해결할 수 있습니다.

sudo nano /boot/config.txt

다음을 추가합니다.

lcd_rotate=2

CTRL+X와 y를 눌러 저장한 다음, 리부트합니다.

sudo reboot


방향 전환까지 하면 완성~

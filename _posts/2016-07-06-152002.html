GitHub Window 쓰다가 Git Shell 을 통해서 계정을 변경하려고 하니까 계속 안 바뀌더라구요

다시 깔고 지우고 하다가 겨우 발견했습니다. 


이 코드를 사용해서 현재의 계정이 누구인지 확인합니다. 


저는 미리 바꿔놓았는데 이런 식으로 뜹니다. 원래 계정을 바꾸기 전에는 이게 아니라 바뀌기 전의 계정이 뜨겠죠?

저의 원래 user.name 이 Mearkat 이었습니다. 

명령어는 다음과 같습니다. (출처 : https://blog.outsider.ne.kr/572 ) 


git config --global --list 
현재 설정정보 조회할 수 있습니다. --global옵션은 전역설정에 대한 옵션이며 현재 프로젝트에만 적용할때는 주지 않습니다.
git config --global user.name "사용자명" 
사용자명을 등록합니다 (필수)
git config --global user.email "이메일주소" 
이메일 주소를 등록합니다. (필수)


이런 식으로 자기가 원하는 사용자 명과 이메일 주소를 입력해줍니다.

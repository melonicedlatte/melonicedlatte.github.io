가끔 git 을 사용하여 코딩을 하다보면 에러가 생길 때가 있다.

이럴 때는 pull 도 되지 않고, push 도 불가능하다.


사실 정확하게 이 파일이 생기는 이유는 잘 모르겠지만 일단 삭제해주고 pull 을 하면 이상없이 작동해서 글을 올린다. 


만약 리눅스에서 사용한다면 리눅스 명령어

rm -f ./.git/index.loc

를 사용하여 제거를 한다. 


윈도우에서는 아래와 같은 명령어를 사용하여 해결할 수 없다. 


순전히 리눅스의 명령어 이기 때문이다. 


윈도우 8.1을 기준으로는 이렇게 해결을 했다.


원래 안드로이드 파일에서 git 은 보이지 않게 설정되어 있다.

이 때, 


위의 탭에서 보기를 누르고 숨긴항목을 클릭한다.





그러면 안 보였던 .git 파일이 보인다.

여기에 들어가서 index.lock 파일을 삭제 해주면 문제가 해결된다.

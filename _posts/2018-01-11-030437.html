> 에러

 CRAN packages 설치에 관련된 문제가 발생합니다. 

ERROR: failed to create lock directory ‘뭐시기뭐시기...../00LOCK-Rcpp’

정확히 어떤 순서로 해결 되었는지는 잘 모르겠지만 아래와 같은 코드들을 실행하다가 되었습니다. 
개인적으로는 R에서 시도해 본 코드가 영향을 준 것 같습니다.  


> 시도 해본 코드

$ sudo apt-get upgrade
$ sudo apt-get update
$ sudo apt-get install r-base-dev
$ sudo apt-get install libnlopt-dev


> R 에서 시도해 본 코드

install.packages("nloptr")
install.packages("설치하고자 하는 패키지 이름", dependencies=TRUE, INSTALL_opts = c('--no-lock'))
install.packages("설치하고자 하는 패키지 이름", INSTALL_opts = c('--no-lock'))

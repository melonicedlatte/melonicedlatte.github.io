lwinV 서버 호스팅 을 하는 업체를 통해서 호스팅을 하는 중입니다.
아마존 보다 합리적인 가격에 좋은 성능을 가졌다고 판단했기때문입니다. 
( 물론 직접 체험해봐야 알 것 같습니다 . 아직 확신은.. ) 

인터페이스 같은 부분은 굉장히 잘 되어 있더라구요. 



서버 종류도 다양하구요 



저는 빨간색으로 선택된 서비스를 사용했습니다. 

서버를 백업해주는 data Storage 10GB 와 IP 가격이 각각 2000원 정도 하는 걸로 보여집니다.
예상 총 가격은 15000원 정도 들 것 같습니다.

아마존 같은 경우에 t2.micro 의 경우 트래픽, 백업, 용량 등을 고려하면 lwinV 가 더 높은 걸로 보여지는데요
정확한 후기는 사용해보고 판단해야겠습니다. 


사용하다보니 처음에는 apt-get 이 작동하지 않았습니다. 

sudo apt-get update 의 명령어도 들지 않구요.

sudo apt-get install ruby


라는 명령어를 사용하면 
ruby 패키지를 사용할 수 없습니다. 하지만 다른 패키지가 참조하고 있습니다.
해당 패키지가 누락되었거나 지워졌다는 뜻입니다. 아니면 또 다른 곳에서
패키지를 받아와야 하는 경우일 수도 있습니다.

라는 명령어가 뜹니다. 

구글링을 통해 판단 한 결과 apt-get 의 서버 주소를 변경해야 된 다는 것을 파악했습니다. 

vim /etc/apt/sources.list 

를 입력합니다. 

그러면 하단부에 아래와 같이 서버가 이상하게 적혀있습니다. 


해당 서버를 아래와 같이 바꾸어줍니다. 

http://ftp.daum.net/ubuntu

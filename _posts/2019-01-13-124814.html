윈도우 10에서 자동 절전하는 문제를 아래에 원본 영어와 번역 글을 써 놓았습니다. 

[english]

When you use windows 10, you can see that suddenly computer go to sleep mode auto. We don't know when computer sleep because windows 10 sleep randomly. You can try below options. 

If your computer is having issues with Windows 10, or Windows 8, going into sleep mode on it's own, this is the answer for you, I have currently had the same issue, Here is a tutorial upon how to fix it. 

1. Click on the windows icon
2. Type regedit
3. Right-click on regedit icon, click Run as administrator
4. Go to HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Power\PowerSettings\238C9FA8-0AAD-41ED-83F4-97BE242C8F20\7bc4a2f9-d8fc-4469-b07b-33eb785aaca0
5. Double click on Attributes
6. Enter number 2.
7. Go to Advanced power settings (click on Windows button, write power options, click on Power Options, in the selected plan click on the Change plan settings, click on the Change advanced power settings).
8. Click on the Change settings that are currently unavailable
9. Click Sleep, then System unattended sleep timeout, then change these settings from 2 Minutes to 20 for example. 


[korean]

윈도우 10 을 사용할 때, 컴퓨터가 자동으로 절전 모드로 들어갈 때가 있습니다. 우리는 윈도우 10이 아무때나 절전을 하기 때문에 언제 컴퓨터가 잠들지 모릅니다. 아래의 옵션을 통하여 해결해보세요. 

만약 윈도우 10이나 8에 컴퓨터 스스로 절전 모드로 들어가는 경우에는, 아래의 옵션으로 고칠 수 있을 것입니다. 


1. 윈도우의 아이콘을 클릭한 다음에 regedit을 입력합니다.
2. 오른쪽 마우스를 눌러서 관리자 모드로 실행하기를 클릭하여 실행합니다. 
3. 해당 경로로 들어가서  HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Power\PowerSettings\238C9FA8-0AAD-41ED-83F4-97BE242C8F20\7bc4a2f9-d8fc-4469-b07b-33eb785aaca0 
4. Attributes를 더블 클릭하고 숫자 2를 입력합니다.



5. 전원 설정에 들어가서 설정해줍니다. (윈도우 아이콘 클릭 -> 전원 및 절전 설정 입력 후 클릭 -> 절전 옵션들을 전부 수정).



6. 추가 전원 설정 -> 선택한 전원 관리에서 설정 변경 -> 고급 전원 관리 설정 변경





7. 전원 옵션에서 하드 디스크, 절전 등에 대한 옵션을 전부 원하는 수치로 변경

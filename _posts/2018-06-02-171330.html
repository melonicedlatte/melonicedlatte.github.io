.gitignore 파일을 생성하면 서버에 바로 적용되지 않는다. 
해당 파일이 남아있는데, 서버에 적용시키기 위해서는 아래의 3개 명령어를 입력해 주면 된다. 

git rm -r --cached .
git add .
git commit -m "fixed untracked files”

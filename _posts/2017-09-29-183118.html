- AMP 설치하기 

Apache2 웹서버, MySQL 데이터베이스 서버, PHP를 Ubuntu에 설치하는 과정을 소개합니다.
amp 을 정리해 놓은 글은 많은데, 그 중에서도 가장 잘 된것 같은 글을 참조해서 저도 포스팅 하고 있습니다.
하단에 링크 참조했습니다. 
글 써주신 분 감사합니다!! 

AMP는 Apache, MySQL, PHP의 첫글자만 가지고 만든 약어입니다.

주로 AMPSetup 이라는 것으로 많은 분들이 윈도우에서 접해 보셨을 것입니다.  

 

1.Apache2 웹서버 설치 

2.MySQL 서버 설치 

3.PHP 설치







- Apache2 설치 

$ sudo apt-get install apache2


위의 명령어로 간단하게 설치할 수 있습니다. 


- MySQL 서버 설치

$ sudo apt-get install mysql-server




설치하다보면 비밀번호 설정 창이 뜨게 됩니다.
비밀번호를 원하는 대로 설정해줍니다. 
설정하면 그 다음 과정이 모두 진행됩니다. 



PHP 설치

PHP는 주로 웹 개발 분야에서 사용되는 스크립트 언어입니다.

여러 가지 버전을 모두 설치한 경우에는 디폴트 PHP 버전 변경 항목을 확인하세요.

 

PHP 패키지를 다운로드 받기 위한 저장소(repository)를 추가합니다.

$ sudo add-apt-repository ppa:ondrej/php


 

설치가 완료되면 apt-get update를 해줍니다. 


$ sudo apt-get update



PHP 7.1

PHP 7.1 기본적인 패키지를 설치합니다.

$ sudo apt-get install php7.1 php7.1-common



이제 추가적으로 필요한 PHP 7.1  패키지를 설치합니다.

$ sudo apt-get install php7.1-mysql php7.1-curl php7.1-xml php7.1-zip php7.1-gd php7.1-mbstring php7.1-mcrypt




아래의 명령어로 php 버전을 확인합니다. 

$ php -v




작동을 확인하면 잘 동작합니다 !!

Jekyll 을 이용하여 요새 github 블로그를 만드는 개발자가 많아지고 있다.
위와 같은 에러 메세지를 출력하면서 설치가 되지 않는 경우가 있다. 
위와 같은 에러가 발생하면 아래와 같은 명령어를 입력해보자.


sudo apt-get install build-essential
sudo apt install ruby-dev make gcc
sudo apt-get install ruby-dev

위의 3가지 명령어를 입력해서 해결해보자.
가장 위의 명령어 순서대로 중요도인 것 같지만 전부 다 입력해서 해결했다. (이래서 docker를 쓰나보다. 다시 공부를 해야겠다..)

윈도우에서는 아나콘다를 이용하여 jupyter notebook 을 사용하고 있는데요

윈도우에서는 생각보다 pip 가 잘 작동하지 않습니다.


리눅스 환경이 개발하기는 더 편한 것 같네요

python 이라는 명령어가 사용되지 않고 제 컴퓨터는 py 라는 명령어가 사용 되더라구요


py -m pip install selenium


위와 같은 명령어를 사용하면 간편하게 사용할 수 있습니다. 

py -m 이라고 명시를 해줘야 pip 가 잘 작동하더라구요


저는 selenium 이라는 라이브러리를 설치했습니다.

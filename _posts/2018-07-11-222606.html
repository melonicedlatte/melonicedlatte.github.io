문제

그룹 단어란 단어에 존재하는 모든 문자에 대해서, 각 문자가 연속해서 나타나는 경우만을 말한다. 예를 들면, ccazzzzbb는 c, a, z, b가 모두 연속해서 나타나고, kin도 k, i, n이 연속해서 나타나기 때문에 그룹 단어이지만, aabbbccb는 b가 떨어져서 나타나기 때문에 그룹 단어가 아니다.

단어 N개를 입력으로 받아 그룹 단어의 개수를 출력하는 프로그램을 작성하시오.

입력

첫째 줄에 단어의 개수 N이 들어온다. N은 100보다 작거나 같은 자연수이다. 둘째줄부터 N개의 줄에 단어가 들어온다. 단어는 알파벳 소문자로만 되어있고 중복되지 않으며, 길이는 최대 100이다.

출력

첫째 줄에 그룹 단어의 개수를 출력한다.

예제 입력 1 

3
happy
new
year

예제 출력 1 

3

예제 입력 2 

4
aba
abab
abcabc
a

예제 출력 2 

1

#include <iostream> 
#include <string>
#include <vector>
 
#pragma warning(disable : 4996)
#define ALPHA 26
 
using namespace std;
 
int main() {
    int TEST_CASE;
    cin >> TEST_CASE; 
    int group_num = 0 ;
 
    for (int test_idx = 0; test_idx < TEST_CASE; test_idx++) {
        string _string; 
        cin >> _string;
 
        vector<int> is_alpha_use(ALPHA, 0);
 
        int is_group = 1; 
 
        // 글자 확인 
        int before_char_int = _string.at(0) - 'a';
        is_alpha_use.at(before_char_int) = 1;
        for (int str_idx = 1; str_idx < _string.length(); str_idx++) { 
            // 만약 이전 글자와 같으면 
            if (before_char_int == _string.at(str_idx) - 'a') continue;
 
            // 이전 글자와 다르고, 이전에 나왔던 글자면? 
            if (is_alpha_use.at(_string.at(str_idx) - 'a')) {
                is_group = 0;
                break;
            }
            // 이전 글자와 다르고, 이전에 나오지 않았다면?
            else {
                before_char_int = _string.at(str_idx) - 'a';
                is_alpha_use.at(before_char_int) = 1;
            }
 
        }
 
        if (is_group) group_num++;
 
    } 
 
    cout << group_num;
    return 0; 
}

## 1. 문제

https://programmers.co.kr/learn/courses/30/lessons/68935

## 2. 풀이

- 3으로 원래 숫자가 1보다 클 동안 나머지 연산을 계속해서 수행하면 3진수의 역수를 구할 수 있다. 이 수를 10진수로 변경하면 정답

~~~python
def solution(n):
    answer = ''
    while n >= 1:
        n, residue = divmod(n, 3)
        answer += str(residue)
    answer = int(answer, 3)
    
    return answer
~~~

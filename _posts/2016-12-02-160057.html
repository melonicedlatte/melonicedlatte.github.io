Github 에 있는 소스코드를 어떻게 활용해야 될까요?? 

Github 에서 코드가 필요할 때가 있습니다. 


모르는 코드를 참고하거나 필요한 소스가 있을 때는 오픈소스나 타인이 올린 프로그램 소스를 가져올 수가 있습니다. 






저는 이번에 알람 소스를 가져와 보도록 하겠습니다.  

cybzz 님이 올려주신 소스가 있네요.


Gradle 이 없는 것으로 보아서는 이클립스에서 만들어진 소스인 것 같습니다.




2014 5월 17일에 최종 업데이트 되었네요.






일단 다운로드를 받은 다음에






프로젝트를 Open 으로 열어보겠습니다. 

위와 같은 구조로 되어있습니다. 






alarmoney-master 를 클릭 해서 오픈해준다. 






그렇게 해서 빌드를 하면 Android 에서는 전혀 보이지 않습니다. 

Android 에서 보이지 않으면 빌드도 되지 않더라구요. 






File -> Import Project 를 해줍니다. 







파일 경로를  ... 을 클릭해서 지정해줍니다.




이 때, alarmoney-master 폴더가 2개가 있는데 libs 와 res 같은 소스들이 있는

바로 위의 alarmoney-master 를 선택해야 합니다.


상위 폴더로 하면 빌드 되지 않습니다. 






Finish 를 해줍니다. 





빌드가 잘 되고...





빌드에 초록불이 들어오고 android 로도 소스가 안보입니다. 





빌드가 잘 됩니다 !!!!

## 1. 문제

https://programmers.co.kr/learn/courses/30/lessons/42840 

## 2. 풀이

- 짝수, 홀수일 때를 구분하여 if 문을 사용하여 다르게 출력해주면 되는 간단한 문제

~~~python
def solution(s):
    return s[len(s)//2] if len(s) % 2 == 1 else s[len(s)//2 -1:len(s)//2 +1]
~~~

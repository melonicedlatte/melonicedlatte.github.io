 <input name="workTime" id="workTime" type="text" class="input">

위와 같은 구조에서는 오른쪽에 화살표 증가, 감소 버튼이 나오게 됩니다.
제거하려면?


가장 간단한 방법은 아래의 css 를 적용하는 겁니다. 

input[type="number"]::-webkit-outer-spin-button,
input[type="number"]::-webkit-inner-spin-button {
    -webkit-appearance: none;
    margin: 0;
}



뜨지 않습니다.

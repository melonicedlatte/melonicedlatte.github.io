위의 링크에 보면 간단한 해결 방법들이 나와있습니다. 


저 같은 경우는 macOS를 사용하고, 아래의 두 명령어를 사용해도 해결되지 않았습니다. 

pip install magic
pip install libmagic


맥에서는 아래와 같은 명령어를 사용하여 해결해줍니다. 

brew install libmagic

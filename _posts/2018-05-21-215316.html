pip 로 파일을 설치했는데 python에서 import 되지 않을 때가 있다.

이는 다양한 버전의 python이 충돌하면서 생기는 문제이다. 


해결책 1. 

~/.bash_profile 파일에 아래의 명령어를 추가해준다. 

export PYTHONPATH="${PYTHONPATH}:/usr/local/lib/python2.7/site-packages:/usr/lib/python2.7/site-packages"



해결책 2. 

python -m pip install

위의 명령어를 사용하여 pip 를 구동하면 기본 설정된 python 으로 설치한다 !!

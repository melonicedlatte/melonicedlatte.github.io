## 3. 소스코드

### 3.1. 내 풀이

~~~python
def solution(arr):
    answer = [arr[0]]
    before_word = arr[0] 
    for x in arr[1:]: 
        if before_word != x:
            before_word = x
            answer.append(x) 
    return answer
~~~

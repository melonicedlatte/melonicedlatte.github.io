## 2. Solve

This is bug on Ubuntu. We have to input under command and solve this problem.

~~~bash
sudo apt-get update -o Acquire::CompressionTypes::Order::=gz
sudo apt-get upgrade -y
sudo apt-get update
~~~

or you can use under command.

~~~bash
vagrant ssh node-01
sudo apt-get clean
sudo apt-get update
~~~

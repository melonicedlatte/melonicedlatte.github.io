제 블로그 같은 경우에는 해외에서 막무가내로 비아그라 광고를 하는 경우가 꽤 있었습니다.

그래서 간혹 db에 접근하여 제가 직접 지워주고 그랬는데요.


그래서 제 호스팅 서버에서 해외 서버의 접속을 아얘 차단했었습니다... 

그걸 잊고 간만에 블로그를 업데이트 하던 도중에 


gem install 을 할 상황이 왔고, 명령어를 실행했습니다. 

$ sudo gem install

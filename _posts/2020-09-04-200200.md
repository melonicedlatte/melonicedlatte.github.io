## 1. 문제

https://programmers.co.kr/learn/courses/30/lessons/12911

## 2. 풀이

~~~python
def solution(n):
    one_num = sum(list(map(int, bin(n)[2:]))) # 2진수로 변환한 뒤에 모든 숫자 더하기
    while True:
        n += 1 
        if one_num == sum(list(map(int, bin(n)[2:]))): # 1의 개수가 같으면 종료
            return n
~~~
